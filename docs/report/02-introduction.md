## 1 引言

### 1.1 研究背景与意义

猕猴桃溃疡病（Kiwifruit Canker）是由丁香假单胞菌猕猴桃致病变种（*Pseudomonas syringae* pv. *actinidiae*, PSA）引起的一种细菌性病害[1]。该病害自1984年在日本首次报道以来，已在全球多个猕猴桃主产区蔓延，造成严重经济损失[2]。感染植株表现为叶片出现水渍状斑点、枝条溃疡、流脓等症状，严重时导致整株死亡[3]。

传统病害诊断主要依赖农业专家的经验性判断，存在以下局限性：（1）主观性强，诊断结果易受个人经验影响；（2）效率低下，难以满足大规模果园的快速筛查需求；（3）实时性差，无法实现早期预警；（4）成本高昂，需要专业技术人员常年巡视。

近年来，随着计算机视觉技术的快速发展，基于深度学习的图像分类方法在植物病害识别领域展现出巨大潜力[4-6]。与传统方法相比，深度学习模型能够自动学习病害的深层特征，实现端到端的自动诊断，显著提高识别效率和准确性。

### 1.2 研究现状与挑战

现有研究多采用标准卷积神经网络（CNN）架构进行植物病害识别。Mohanty等[7]使用AlexNet和GoogLeNet对26种作物病害进行分类，准确率达到99.35%。Ferentinos[8]比较了多种CNN模型在植物病害识别中的性能，发现VGG和Inception-v3表现最佳。

然而，现有方法在猕猴桃溃疡病识别中仍面临以下挑战：（1）病害特征与健康组织差异细微，属于细粒度分类问题；（2）田间采集图像背景复杂，存在光照变化、遮挡等干扰；（3）可用标注数据有限，属于小样本学习问题。

注意力机制通过选择性增强重要特征、抑制无关信息，在细粒度图像分类任务中表现出优越性能[9]。Hu等[10]提出的Squeeze-and-Excitation（SE）模块通过建模通道间依赖关系，显著提升了图像分类性能。但在植物病害识别领域，特别是在猕猴桃溃疡病检测中，SE模块的应用效果尚待深入研究。

### 1.3 本文贡献

本文的主要贡献如下：

1. **提出改进模型**：将SE注意力模块融入ResNet34架构，构建SE-ResNet34模型，增强模型对病害关键特征的感知能力。

2. **构建专用数据集**：收集并标注包含600张图像（健康与患病各300张）的猕猴桃叶片数据集，为后续研究提供数据支持。

3. **系统实验验证**：通过对比实验、消融实验和可视化分析，全面验证所提方法的有效性。

4. **深入机理分析**：利用Grad-CAM等方法可视化模型注意力，从机理层面解释SE模块的作用。
