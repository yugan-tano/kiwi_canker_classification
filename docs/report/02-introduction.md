# 1. 引言

## 1.1 研究背景

猕猴桃溃疡病（Kiwifruit Canker）是由*Pseudomonas syringae* pv. *actinidiae*（PSA）细菌引起的一种毁灭性病害，自1980年代首次爆发以来，已在全球多个猕猴桃主产区造成严重经济损失[1]。该病害主要危害猕猴桃叶片、枝条和主干，导致植株生长受阻、产量下降甚至整株死亡。

传统的病害诊断主要依赖农业专家的人工目视检查，这种方法存在以下局限性：
1. **主观性强**：诊断结果易受个人经验影响；
2. **效率低下**：大规模果园检查耗时耗力；
3. **实时性差**：难以及时发现早期病害；
4. **成本高昂**：需要专业技术人员常年巡视。

## 1.2 研究意义

随着计算机视觉和深度学习技术的飞速发展，基于图像的植物病害自动识别成为智慧农业领域的研究热点。本研究的意义主要体现在：

**理论意义**：
- 探索注意力机制在农业图像细粒度分类任务中的应用效果；
- 验证SE模块与传统CNN架构结合的性能增益；
- 为小样本农业图像分类提供技术参考。

**实践价值**：
- 开发可部署的自动化病害诊断原型系统；
- 降低果园管理成本，提高病害防治效率；
- 为农民提供便捷、低成本的病害识别工具；
- 促进精准农业和农业信息化发展。

## 1.3 研究目标

本研究的主要目标包括：
1. 构建一个包含健康与患病猕猴桃叶片的图像数据集；
2. 实现基于SE-ResNet34的改进分类模型；
3. 通过对比实验验证模型的有效性；
4. 分析SE注意力机制在病害识别中的具体作用。
