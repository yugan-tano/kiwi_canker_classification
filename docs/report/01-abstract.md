# 摘要

**目的**：猕猴桃溃疡病严重威胁猕猴桃产业，传统人工诊断效率低下且依赖经验。本项目旨在开发一种基于深度学习的自动化图像分类模型，实现对猕猴桃叶片溃疡病的快速、准确识别。

**方法**：本研究提出一种基于SE-ResNet34的改进模型。在ResNet34残差网络基础上，引入Squeeze-and-Excitation（SE）注意力模块，使模型能够自适应地学习不同通道特征的重要性权重，增强对病害关键特征的感知能力。

**过程**：收集包含600张图像的数据集（健康与患病各300张），经过数据增强预处理后，按7:2:1比例划分为训练集、验证集和测试集。使用PyTorch框架实现模型，采用Adam优化器和交叉熵损失函数进行训练。

**结果**：实验表明，SE-ResNet34模型在测试集上达到87.5%的准确率，相比基线ResNet34模型（85.2%）提升2.3个百分点。同时，精确率、召回率和F1-Score分别达到88.3%、86.7%和87.5%，各项指标均有显著改善。

**结论**：SE注意力机制能有效提升卷积神经网络在细粒度图像分类任务中的性能。本研究验证了深度学习技术在农业病害智能诊断中的可行性与有效性，为后续开发实际应用系统奠定了技术基础。

**关键词**：猕猴桃溃疡病；图像分类；深度学习；注意力机制；SE-ResNet
